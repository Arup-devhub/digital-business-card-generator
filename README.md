# Digital Business Card Generator 

A fast, responsive web application built for college students to generate professional digital business cards on the fly. No backend required—built entirely with client-side technologies.

## Tech Stack
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript
* **Library:** [html2canvas](https://html2canvas.hertzen.com/) for DOM-to-Image rendering

## Key Features
* **Live Preview:** Real-time data binding; see your card update as you type.
* **Template Engine:** Toggle between Classic, Modern Dark, and Cyber Neon themes.
* **Instant Export:** Download your custom card as a high-quality PNG file.
* **Responsive Design:** Works flawlessly on desktop and mobile devices.
